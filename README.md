# Video Game Sales Analysis

## Overview
This project explores historical global video game sales data to identify business insights about platforms, genres, publishers, regions, and yearly sales trends.

The goal is to act as a data science consultant for a video game company and use data analysis to support decisions about game development, platform strategy, and regional marketing.

## Business Problem
A video game company wants to understand which video game genres, platforms, publishers, and regions achieve the highest sales.

This analysis helps answer questions such as:

- Which genres generate the highest global sales?
- Which platforms perform best?
- Which publishers dominate the market?
- How do sales differ across North America, Europe, and Japan?
- How have video game sales changed over time?
- Which platform and genre combinations are strongest?

## Dataset
Dataset: Video Game Sales Dataset  
Source: Kaggle

The dataset includes historical video game sales information such as:

- Game name
- Platform
- Year of release
- Genre
- Publisher
- North America sales
- Europe sales
- Japan sales
- Other region sales
- Global sales

## Methods
The project includes:

- Data loading and inspection
- Missing value analysis
- Duplicate checks
- Data cleaning
- Exploratory data analysis
- Grouped analysis by genre, platform, publisher, region, and year
- Trend analysis
- Business interpretation and recommendations

## Key Insights
The analysis shows that certain genres, platforms, and publishers have historically performed better than others.

Regional differences are also important. North America, Europe, and Japan show different sales patterns, which means that launch and marketing strategies should consider regional preferences.

The project also shows that platform and genre combinations can be key drivers of video game sales.

## Business Recommendations
A video game company should consider developing games in genres and platforms that have historically performed well.

The company should also consider regional differences when planning marketing campaigns and launch strategies.

Because the dataset is historical, the insights should be combined with newer market data before making final business decisions.

## Limitations
- The dataset is historical and may not fully represent the current gaming industry.
- The dataset focuses mainly on sales and does not include other useful features such as marketing spend, user reviews, game quality, or digital subscription data.
- Future analysis should include more recent gaming market data.
